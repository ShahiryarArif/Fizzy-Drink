# Fizzy-Drink
# This project uses the Alpino font from Fontshare. You can find more information about the font and its licensing at [Fontshare's website](https://www.fontshare.com/fonts/alpino).
# This project utilizes SVGOMG for optimizing SVG files. You can access SVGOMG at [SVGOMG's website](https://jakearchibald.github.io/svgomg/).
# svg2jsx is used in this project for converting SVG files to JSX format. You can find more information about svg2jsx at [svg2jsx's GitHub repository](https://github.com/svgr/svgr).
