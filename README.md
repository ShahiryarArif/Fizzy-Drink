# Fizzy-Drink
# Using Alpino font from Fontshare. You can find more information about the font and its licensing at [Fontshare's website](https://www.fontshare.com/fonts/alpino).
# SVGOMG for optimizing SVG files. You can access SVGOMG at [SVGOMG's website](https://jakearchibald.github.io/svgomg/).
# SVG2JSX is used in this project for converting SVG files to JSX format. You can find more information about svg2jsx at [svg2jsx's GitHub repository](https://github.com/svgr/svgr).
# GSAP (GreenSock Animation Platform) is used for creating animations in this project. You can find more information about GSAP at [GSAP's official website](https://greensock.com/gsap/).
# npm install gsap @gsap/react
# Prismic CMS is used for content management in this project. You can find more information about Prismic at [Prismic's official website](https://prismic.io/).
# npm install @prismicio/client @prismicio/react @prismicio/next
# React Three Fiber is used for 3D rendering in this project. You can find more information about React Three Fiber at [React Three Fiber's official website](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction).
# npm install three @types/three @react-three/fiber 
# npm install @react-three/drei
# npm i -D r3f-perf
